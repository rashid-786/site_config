USE CASE
===================
SITE INFORMATION FORM
* Login with admin role.
* Go to admin/config/system/site-information
* Check the textfield with name "Site API Key" and with default value as 
 "No API Key yet".
* Change the default value with any values and click on update configuration 
  button.
* See the changes with drupal status message.

PAGE URL WITH JSON DATA OR ACCESS DENIED 
* To check the URL Go to /page_json/%API_KEY/%NODE_ID.
